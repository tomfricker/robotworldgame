Collections

In this level, the concept of collections are introduced. Collections are a convenient way of storing data in programming. Java uses a range of different collections to store data in different ways. Common collections are:

  • ArrayList
  • HashMap
  • HashSet
  • LinkedList

Collections vary mostly in structure but a user often needs to concern themselves with only a few of their methods.
The two most common features of collections are adding and removing objects to and from them.
the majority of collections will use the add() method like the one featured in the collections level. However HashMaps use put() instead.

The type of object you can put into a collection is determined when it is instantiated in code.
The four collection types listed above have the following declarations:

  • ArrayList<{objectType}>
  • HashMap<{Key},{objectType}>
  • HashSet<{objectType}>
  • LinkedList<{objectType}>

The {objectType} part of the declaration is replaced with any type. For example, an ArrayList<Integer> (note the use of the wrapper type Integer rather than primitive type int) can only be used to store int values.

The HashMap collection is slightly different. It uses a Key which relates to a specific object of the objectType in the Map. the type of the key can be different to that of the Objects being stored e.g. <Integer,String> but they may also be the same e.g. <Integer, Integer>.
There can not be more than one instance of the same key used, however, multiple values may be the same.
i.e. you can have <1,"Hello"> and <2,"Hello">
but you can not have <1,"Hello"> and <1,"Goodbye">.

in the latter case, only the second key value pair put into the map will remain at the end.

Playing the Level

The Level has two stages.

Stage 1

Firstly, three collections of different types will ask you to collect their respective data types which are spread over the board.
You need to talk to the them to be able to collect their objects. and you can only collect for one Collection at a time.

When you have collected the object for the Collection, you need to return to them to turn your pickups in, in return for points.

Once you collect all of the Pickups and return them to their collections, you will progress to Stage 2.

Stage 2

In stage 2, you will meet a lone collection. You need to use the Code Panel to add char (character) objects to the collection.

The collection's name is chars.
