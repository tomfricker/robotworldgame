Loops

There are three different types of loops that you need to know about.

1) for each

This loop is ONLY used for collections and has the following syntax:

for(ElementType element : collection) {

	loop body

}

The for each loop carries out the loop body for each element in the collection.

  • ElementType element - The ElementType is what class the objects in the collection are, and the element is a reference to an object in the collection. You choose what element is called.
  • collection - a collection of objects of the class ElementType.
  • loop body - statements which will be repeated for each element in the collection.

Here is the example from the game: 

for(Flower flower : flowers) {

	pickup(flower);

}

First we will look at each part of the loop.

  • Flower flower - The element flower is an object of teh class Flower. Here I chose the element to be called flower.
  • flowers - a collection of flowers (where each is of the class Flower).
  • pickup(flower); - each flower in the collection flowers will be picked up.

So, imagine you have a numbered list of flowers. This loop will start with the first flower on your list and pick it up. Then it will move on to the second and pick that one up, and so on, until all the flowers on your list have been picked up.



2) while

This loop is used when you want to repeat the loop body an unknown number of times.
It has the following syntax:
Note: boolean means it is either true or false. 

while(boolean condition) {

	loop body

}

The while loop carries out the loop body until the boolean condition is false.

  • boolean condition - an expression that can be either true or false.
  • loop body - statements which will be repeated.

Here is the example from the game: 

int flowerIndex = 0;

while(flowerIndex < flowers.size()) {

	pickup(flower);

	flowerIndex ++;

}

First we will look at each part of the loop.

  • int flowerIndex = 0; - First we initialise a variable flowerIndex which is an integer and is equal to zero.
  • flowerIndex < flowers.size() - the loop will continue while the flowerIndex is less than the number of flowers.
  • pickup(flower); - the flower will be picked up.
  • flowerIndex ++; - this will add one to the flowerIndex.

So, once again, imagine you have a numbered list of flowers which starts with zero. This loop will start with whatever flower is at the flowerIndex number, it will pick up that flower and then add one to the flowerIndex. Then it will move on to the flower at the new number of the flowerIndex and pick that one up, and so on, until flowerIndex is a bigger number than the number of flowers available.



3) for

This loop is used when you want to repeat the loop body an known number of times.
It has the following syntax: 

for(initialization; loop condition; increment/decrement) {

	loop body

}

The for loop carries out the loop body until the loop condition is false.

  • initialization - is done once at the beginning.
  • loop condition - an expression that can be either true or false.
  • increment/decrement - is done after the loop body.
  • loop body - statements which will be repeated.

Here is the example from the game: 

for(int i = 0; i < 3; i++) {

	pickup(flower);

}

First we will look at each part of the loop.

  • int i = 0; - First we initialise i which is an integer and is equal to zero.
  • i < 3; - the loop will continue while i is less than 3.
  • i++ - add one to i.
  • pickup(flower); - the flower will be picked up.

So, you might have noticed that this loop does exactly the same as the while loop example, but here we know how many flowers we want the loop to repeat for.


(Note: that all of these examples are for demonstrating how a loop works and the loop bodies will not work as written here.)
