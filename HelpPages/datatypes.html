Data Types

In this level you will learn about how data is stored in a piece of software.

The following are the stages of the level.

  • Stage 1 - Introduction
  • Stage 2 - Integers
  • Stage 3 - Strings
  • Stage 4 - Arithmetic and End Quiz

The information below should help you with the questions that you are asked in the level.

The following is a list of primitive data types. Notice they all start with a lowercase letter.

  • int - A whole number. e.g. 10
  • char - A single character, denoted by ' '. e.g. 'a'
  • boolean - Something that is true or false. e.g. true
  • double - A fractional number. e.g. 10.321

Below are some examples of data types that belong to a class. Notice they all start with an uppercase letter.

  • String - A series of characters denoted by " ". e.g. "robots"
  • ArrayList - A collection of objects. This is a class that is pre-built into Java. You will learn more about these in level 3.
  • Game - A programmer defined class. One that is designed by the programmer for use within a piece of software, you will learn more about these in level 2.

The following are examples of different declarations of data types.

  • int age = 60;
  • char one = 'a';
  • boolean isLevelOne = true;
  • boolean isLevelTwo = false;
  • String name = "Fernando";
  • int x = 4;
  • int y = 5;
  • int z = x + y;

Always remember the order of a declaration is:

  • Data type - i.e. int
  • Variable name - i.e. buttons
  • Assignment statement - =
  • Data - i.e. 3
  • Semi-colon - ;
