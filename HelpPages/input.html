How to use the Input Panel

One command:

  • Enter the command in the small box.
  • Press Run code.

Multiple commands:

  • Enter the command in the small box.
  • Press the Enter key on your keyboard.
  • Repeat until you have entered all of your commands.
  • Press Run code.

Reusing commands:

If you want to use a command you have typed before, instead of retyping it, press the up arrow until the command you want appears.
