Stage 1: Relational Operators

In java there are 6 relational operators which are listed below with their symbol in brackets: These operators are used to compare two values. The signs below except for the equals operator == are used to show inequality between two values.

1. Greater than (>): This is used to show that the first value is greater than the second value e.g. 5 > 2

2. Less than (<): This operator is used to show that the first value is less than the second value e.g 2<5

3. Equals (==): Do not mistake this for the assignment operator (=). This is the only sign that checks for equality and is used usually when two values are the same e.g 2 == 2

It does not necessarily have to be used with integer values. They can also be used with other primitive data types like boolean, String , char, double etc.
However in this level we will be looking at how we can apply this operator and the others to boolean values since this is important if you are going to understand how conditional statements work.

4. Not Equal (!=): this sign checks to see if both values are not equal to the other. 7 != 4 since 7 and 4 are not equal to each other this would result in true. If the values did equal to each other then it would result in false.

5. Greater than or equal to (>=) : This operator is similar to the greater than operator in that it checks to see if the first value is bigger than the second but in java when it is combined with an equals symbol it also checks if the first value is equal to the second.

6. Less than or equal to (<=): Just like the greater than or equal to operator this one is used not only to check if the first value is less than the second but also if it is equal or not.

Note: The first value can be either less than or equal to, it doesn't have to be both to result in true. This is the same for the greater than or equal to operator.

Stage 3: Conditional Statements

Conditional statements help the computer make decisions to do things.

They include a boolean condition which is checked to see if it evaluates to true or false.
If it is true, then a certain part of the code will be executed.
The most common conditional statements are called if then statements.

This is an example of how an if then statement is written:


If then statement syntax 

If (condition) {

	// then do this

}

Other conditional statements that are used are called If else statements. These offer an alternative decision if the condition evaluates to false. 



If else statement syntax

If (condition) {

       // then do this

} else {

      // do this }

  }

}
